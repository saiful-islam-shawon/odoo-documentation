domain & env

domain : domain is used to search value in a model.

patient_id = fields.Many2one(
    'hospital.patient',
    string='Patient',
    domain=[('state', '=', 'confirmed')],  # only show confirmed patients
)

eve : env give me access to everything. we can find anything from a model by end
what env can do

Property													What it gives you
`self.env['model.name']`									Access any model
`self.env.user`												Current logged in user
`self.env.uid`												Current user ID
`self.env.company`											Current company
`self.env.cr`												Database cursor
`self.env.context`											Current context dictionary

Most Common Uses

  1. Access Another Model
# Get all patients
patients = self.env['hospital.patient'].search([])

# Get confirmed patients only
patients = self.env['hospital.patient'].search([
    ('state', '=', 'confirmed')
])
