#: odoo te always for rec in self: ata kno use kora hoi
Answer: Odoo তে self মানে সবসময় একটা record না, একটা recordset।

# self হতে পারে —
self = product.pricelist(1)           # একটা record
self = product.pricelist(1, 2, 3)     # তিনটা record
self = product.pricelist()            # খালি recordset

#: self.env.ref() ar kaj ki
Answer : self.env.ref() দিয়ে XML ID থেকে record খোঁজা হয়।
Odoo তে প্রতিটা record এর একটা unique XML ID থাকে। যেমন:

<record id="mail_activity_data_todo" model="mail.activity.type">

এই record টা database এ আছে, কিন্তু Python এ access করতে হলে self.env.ref() ব্যবহার করো:

activity_type = self.env.ref('mail.mail_activity_data_todo')
# mail = module name
# mail_activity_data_todo = record এর id

#: filter and lambda when use kora hoi
Answer: filtered() with lambda ব্যবহার করা হয় যখন recordset থেকে condition অনুযায়ী specific records বের করতে হয়।

approver = rec.approve_user_ids.filtered(lambda a: a.user_id.id == current_user.id)

approve_user_ids তে অনেক user থাকতে পারে, সেখান থেকে শুধু current user এর record বের করছে।
